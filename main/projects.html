PantherView

PantherView is a collaborative Computer Science Club Project in HTML, CSS, and Javascript utilizing Leaflet.js with OpenStreetMap. Retrieves data from the Western Pennsylvania Regional Data Center such as arrest and code violation records as well as data from a custom Pitt API (written in Python, hosted as a Flask application on AWS). The data is then organized into filterable pins that are placed on a map to show locations of incident records, or information about Pitt services.

Cluster Computing Research with ClusterHAT

Current project in Advanced Systems Software using a Raspberry Pi 3 and a HAT (Hardware Attachment on Top) that allows it to interface with up to four Raspberry Pi Zeros. The purpose is to and design create a distributed computing infrastructure that fits in the palm of your hand, as well as a performance testing suite that measures performance with different configurations and workloads.
